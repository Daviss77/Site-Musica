O intuito desse site foi melhorar a prática no HTML, CSS e JavaScript. Pois, na faculdade iremos usar bastante essas ferramentas e linguagem de programação.
O Site / Blog, consiste em uma ideia que todos nós gostamos na área de tecnologia, que seria a música. Então resolvemos trazer essa ideia para vocês, cada um de nós escolhemos 2 gêneros musicais para falar sobre, sendo eles (MPB, SAMBA, JAZZ, BLUES, GRIME, FUNK) no todo 6 gêneros, elementos que tem histórias bem legais e grandes marcas na sociedade.
Espero que vocês gostem dessa proposta e absorvam algum conhecimento dessa trilogia musical e se sintam imersos.
Uma breve explicação: Na tela Menu, irá direcionar vocês para onde querem, caso queira ver os desenvolvedores basta clicar no (DEV'S), caso queira ir para a escolha dos gêneros clique em (GÊNEROS) ou queira fazer o cadastro para saber de uma futura atualização, clique em (CADASTRO). Obrigado :) tenha uma boa experiência.
